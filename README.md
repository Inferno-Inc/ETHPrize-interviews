# ETHPrize

#### Install instructions
###### Install all the project dependencies
```>$ npm install ```

###### To run a local development server
```>$ npm run dev```

###### To run a local development server with browser syn
```>$ npm run watch ```

###### Build the code  - Production (Uglify and minimise)
```>$ npm run production ```
